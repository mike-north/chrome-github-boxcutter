# GitHub Boxcutter

Better package manifests on Github
Boxcutter enhances package manifests on github. 

* Click on package names to jump to their repositories

Support for the following package managers (more coming soon):

* NPM (JavaScript)
* Bower (JavaScript)
* Gem (Ruby)
* Hex (Elixir)
